# PE文件分析与压缩壳实现 系列课程

## 课程简介

本系列课程基于 stub-pack 项目，系统讲解 Windows PE 文件结构、压缩壳原理与实现。通过本课程学习，你将深入理解可执行文件的底层机制，掌握构建简单压缩壳的完整流程。

## 目标受众

- 对二进制分析感兴趣的安全爱好者
- 逆向工程初学者
- Windows 程序开发人员
- 恶意代码分析入门者

## 前置知识

- C/C++ 基础
- 计算机组成原理
- 汇编语言基础
- Windows 操作系统基础

## 课程结构

### 第一部分：PE文件基础
1. PE文件结构概述
2. DOS头与NT头详解
3. 节表与节属性
4. 数据目录详解

### 第二部分：PE文件高级特性
1. 导入表与导出表
2. 重定位表
3. TLS表
4. PE文件加载流程

### 第三部分：压缩壳原理
1. 加壳技术概述
2. 压缩算法基础
3. 自解压原理
4. stub代码设计

### 第四部分：压缩壳实现（基于stub-pack项目）
1. 项目架构解析
2. PE文件解析与重建
3. 数据压缩实现
4. 自解压代码实现
5. 项目实战与优化

## 教学资源

本课程配套以下教学资源：
- 📚 **学习笔记**：详细的知识点讲解与代码分析
- 📊 **PPT课件**：课堂教学用演示文稿
- 💻 **源代码**：stub-pack项目完整代码
- 🎯 **实战练习**：各章节配套练习题

## 学习路径

```
基础知识 → PE文件结构 → 加壳原理 → 实战实现 → 进阶优化
```

## 预期学习成果

完成本课程后，你将能够：
1. 深入理解PE文件结构
2. 分析和操作PE文件
3. 理解压缩壳的工作原理
4. 实现一个简单的压缩壳
5. 为进一步学习逆向工程和恶意代码分析打下基础

## 学习建议

1. **理论与实践结合**：每学完一个章节，立即动手实践
2. **工具使用**：熟悉PE分析工具（如PEiD、CFF Explorer、x64dbg等）
3. **代码分析**：仔细阅读和理解stub-pack项目代码
4. **实验验证**：使用调试器观察加壳程序的运行过程
5. **扩展阅读**：参考推荐的进阶阅读材料
